### Hi, I'm Ankit 👋

I'm a data engineering leader working on cloud-native, real-time data platforms, and someone who still loves getting hands-on with code through personal projects on the side.

#### 💼 What I do

I architect real-time, high-throughput data pipelines that turn raw data into fast, reliable decisions, built on AWS and Kafka. My focus has increasingly shifted toward Generative AI — building RAG pipelines, LLMOps workflows, and agentic AI features that ship in production for large-scale users. I'm an AWS Certified Solutions Architect – Associate, and completed a postgraduate specialization in Machine Learning & AI at IIIT Bangalore.

#### 🚀 Side projects I build for fun

**[skopelabs](https://github.com/AnkitSurana/skopelabs)** — An explainable computer vision engine that scans video and live camera feeds for reference images, using classical techniques like ORB feature matching and RANSAC rather than black-box models.

**[sutham-app](https://github.com/AnkitSurana/sutham-app)** — A mobile app that guides households through sorting waste correctly using live camera input, working across languages and offline.

**[collabdocs](https://github.com/AnkitSurana/collabdocs)** — A Django REST Framework backend for shared workspaces, document collaboration, and team permissions.

**[devtrack](https://github.com/AnkitSurana/devtrack)** — A backend API for tracking engineering issues, priorities, and status.

**[eventhub](https://github.com/AnkitSurana/eventhub)** — A backend API for an event ticketing platform, handling browsing, reservations, and cancellations.

**[applyin-backend](https://github.com/AnkitSurana/applyin-backend) & [applyin-extension](https://github.com/AnkitSurana/applyin-extension)** — A backend service paired with a Chrome extension that streamlines job applications.

**[Melanoma Detection using CNN](https://github.com/AnkitSurana/AnkitSurana-Melanoma-Detection-Using-CNN)** — A convolutional neural network trained to classify skin lesion images.

**[Credit Card Fraud Detection](https://github.com/AnkitSurana/Credit-Card-Fraud-Detection)** — A machine learning model built to flag fraudulent transactions.

**[BoomBikesSharing](https://github.com/AnkitSurana/BoomBikesSharing)** — Demand forecasting for shared bikes using regression modeling.
